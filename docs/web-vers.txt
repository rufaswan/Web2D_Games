Desktop vs Mobile
	https://wiki.mozilla.org/Mobile/Platforms
	Firefox for Android is built on the same engine as Firefox desktop,
	with extensive changes to optimize performance and user experience on mobile devices.

	https://developer.chrome.com/docs/multidevice/faq/
	Chrome for Android is derived from Chromium.
	They share the same release cycle and release numbering.
	Not2 all features in desktop Chrome are available in the corresponding version of Chrome for Android, however.

	https://developer.apple.com/documentation/safari-release-notes
	Safari is a web browser app and web technology platform available on iOS and macOS.
	It’s built on WebKit, a fast, open-source web rendering engine that implements web standards.

	https://developer.chrome.com/docs/multidevice/webview/
	Chrome for Android is separate from WebView.
	They're both based on the same code, including a common JavaScript engine and rendering engine.
